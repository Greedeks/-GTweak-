 A small utility to customize Windows to your needs.

## A small list of features
- Windows Activation
- Disabling Windows Defender
- Disabling unnecessary services
- Interface customization
- Changing the power supply circuit
- Removing UWP Apps
- Deleting OneDrive
- Disabling Ads

 A small utility to customize Windows to your needs.

## A small list of features
- Windows Activation
- Disabling Windows Defender
- Disabling unnecessary services
- Interface customization
- Changing the power supply circuit
- Removing UWP Apps
- Deleting OneDrive
- Disabling Ads

<b><i>You will have to stop the antivirus before using it</i>

## Небольшой список возможностей
- Активация Windows
- Отключение защитника Windows
- Отключение ненужных служб
- Кастомизации интерфейса
- Измение схемы электропитания
- Удаление UWP приложений
- Удаление OneDrive
- Отключение рекламы

<b><i>Перед использованием придется остановить  антивирус</i>
